如果要查看built-in的效果，要删除 SRP Setting里的配置
如果要查看URP的效果，就要设置 SRP Settings，如果工程中没有，可以在Project中右键，Create -> Rendering -> URP Asset(with Universal Renderer)
SRP Setting的具体路径为: Edit -> Project Settings -> Graphics -> Scriptable Render Pipeline Settings

对于URP还有一个URP Global Settings，主要是针对 Rendering Layers和Shader Stripping进行设置
